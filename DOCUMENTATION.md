## 1. 项目概述

### 1.1 项目简介

MuMuAINovel（木木AI小说）是一个基于 AI 的智能小说创作助手，帮助用户在 AI 辅助下创作小说，包括生成大纲、角色、章节和世界观设定。

### 1.2 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | FastAPI + SQLAlchemy Async + Alembic |
| 数据库 | PostgreSQL 18 / SQLite |
| 前端 | React 18 + TypeScript + Ant Design 5.x + Zustand |
| AI 服务 | OpenAI / Anthropic Claude / Google Gemini |
| 向量检索 | sentence-transformers + ChromaDB |
